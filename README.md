## How To Use

```bash
# Clone this repository
$ git clone https://github.com/gustavomartins95/crud-knex

# Go into the repository
$ cd crud-knex

# Install dependencies
$ npm install

# Run the app
$ npm run start
```

To view the project you can open http://localhost:3000.
